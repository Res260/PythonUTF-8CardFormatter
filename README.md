# PythonUTF-8CardFormatter
A small-ish function to transform a list of strings to a single string that's actually UTF-8 cards of said strings.

Basically, all it does is transform this:

["Notre vie avant le Cégep", "Une assiette chinoise", "Être toujours en retard", "CopyPasta", "TRIGGERED", "Google Doc", "Ta mom", "Le dad à Gonzo", "Le tapis Daem", "Le garage à Gonzo"]
